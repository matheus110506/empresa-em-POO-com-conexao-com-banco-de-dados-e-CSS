# Cadastro de Produtos

Sistema web para cadastro e listagem de produtos.

Tecnologias:
- PHP
- HTML
- CSS
- MySQL

Funcionalidades:
- Cadastro de produtos
- Listagem de produtos

O usuário começa no menu principal, onde há 2 botões: um para cadastrar produtos e outro para consultar uma tabela onde eles estão listados. Ao cadastrar um produto, são inseridos o código, nome, data de validade e preço do produto. Esses dados serão armazenados no banco de dados e na tabela da página de consulta. É possível retornar ao menu e acessar a página da consulta dos produtos cadastrados.
